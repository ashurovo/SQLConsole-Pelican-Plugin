# SQL Console for Pelican Panel

SQL Console adds a lightweight and fully integrated database query interface directly to your Pelican server management panel for quick maintenance tasks, debugging, or data retrieval.

## Usage

1.  Go to the **SQL Console** tab in your server sidebar.
2.  Select the **Target Database** from the dropdown.
3.  Type your SQL query (e.g., `SELECT * FROM users`).
4.  Click **Run Query**.
5.  (Optional) Click **Download CSV** to save the results.
